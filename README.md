# Gerador de Relatórios Executivos PMO (VBA para PowerPoint)

Este projeto automatiza a criação de Dashboards executivos, extraindo dados de uma tabela de projetos no Excel e gerando slides profissionais no PowerPoint com um único clique.

## 🚀 Funcionalidades
- **Interface Intuitiva:** UserForm customizado para seleção de filtros.
- **Filtros Dinâmicos:** Opções de visualização para 30, 60, 90 dias ou Pipeline completo.
- **Design Clean:** Gráficos de barras e tabelas dinâmicas sem bordas escuras, seguindo padrões de design executivo.
- **Análise Multidimensional:** Slides focados em Status de Portfólio, Composição por Tipo e Análise Geográfica/Bandeira.

## 🛠️ Como usar
1. Baixe o arquivo na pasta `/template`.
2. Certifique-se de que sua tabela de dados se chama `tbProjetos`.
3. Abra o formulário através do botão na planilha.
4. Selecione o período desejado e clique em "Gerar".

## 💻 Tecnologias
- Excel VBA
- PowerPoint Object Library
